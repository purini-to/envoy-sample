# envoy-sample

envoy sample in golang

## Quick Start

```bash
git clone https://github.com/purini-to/envoy-sample.git
cd envoy-sample
make compose
```

service access to 
echo: http://localhost, http://localhost/private, http://localhost/tasks, http://localhost/tasks/{id}  
jaeger access to http://localhost:16686
